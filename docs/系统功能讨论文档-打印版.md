## 一、系统现状简述

FB Auto Analytics 是一套 Facebook 广告数据分析平台，已与 Shoplazza 店铺 UTM 数据打通，支持广告三层结构（系列 / 广告组 / 广告）的指标展示与 UTM 衍生指标计算。

**当前技术架构要点：**

- 前端：React + Ant Design，主要页面包括数据仪表盘、广告管理、批量发布、店铺 Token、店铺映射
- 后端：Node.js + Express，定时同步 Facebook 与 Shoplazza 数据至 PostgreSQL
- 数据策略：广告管理页采用「DB-First」——前端读库，后台异步同步（UTM 5 分钟 / 指标 15 分钟 / 结构 6 小时）
- 匹配规则：`fb_ads.ad_id` = `shoplazza_utm.utm_value`（dimension = utm_content）

**当前业务规模（参考）：**

- 广告账户：约 35 个
- 店铺：4 个
- 预期同时在线用户：10–20 人
- 计划部署：美国服务器

### 3.1 指标计算公式

| 前端指标 | 计算公式 | 数据来源 |
|----------|----------|----------|
| 单次连接点击花费 | spend ÷ uv | FB spend / Shoplazza uv |
| 单次成效花费 | spend ÷ orders | FB spend / Shoplazza orders |
| ROAS | sales ÷ spend | Shoplazza sales / FB spend |
| 单次加购费用 | spend ÷ add_to_cart | FB spend / Shoplazza add_to_cart |
| 单次结账费用 | spend ÷ begin_checkout | FB spend / Shoplazza begin_checkout |
